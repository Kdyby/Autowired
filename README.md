Requirements
------------

Kdyby/Autowired requires PHP 8.0 or higher.

- [Nette Framework](https://github.com/nette/nette)


Installation
------------

The best way to install Kdyby/Autowired is using  [Composer](http://getcomposer.org/):

```sh
$ composer require kdyby/autowired
```
